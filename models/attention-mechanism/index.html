Attention History

Attention mechanisms emerged in neural machine translation (Bahdanau et al., 2015) and were revolutionized by the Transformer (Vaswani et al., 2017), which used self-attention exclusively.

Attention Implementation Tips

Use PyTorch's MultiheadAttention or implement custom attention. Scale dot products by √d_k. Apply dropout to attention weights. Use causal masking for autoregressive tasks. Visualize attention weights to validate learning.

Frequently Asked Questions

What is the difference between attention and self-attention?

Attention typically refers to cross-attention between different sequences (e.g., encoder-decoder). Self-attention refers to attention within the same sequence, where queries, keys, and values all come from the same input.

Why is scaled dot-product attention used?

Scaled dot-product attention divides by √d_k to prevent the dot products from growing too large, which would push the softmax into regions with extremely small gradients. This stabilizes training.
