NIG Model: Normal Inverse Gaussian Pricing

Master Normal Inverse Gaussian process for option pricing with heavy tails and skewness. Learn NIG characteristic functions and financial applications.

Quick Answer

NIG process: X_t = μt + β·IG_t + √(IG_t)·W_t where IG_t is Inverse Gaussian. Pure jump process with normal inverse Gaussian margins. Captures skewness and heavy tails.

NIG Distribution Formula

NIG Probability Density:

f(x) = (αδ/π) · exp(δ·√(α² - β²) + β(x-μ)) · K₁(α√(δ² + (x-μ)²)) / √(δ² + (x-μ)²)

Where:
K₁ = Modified Bessel function of second kind
α > 0 = Tail parameter
|β| < α = Asymmetry parameter
δ > 0 = Scale parameter
μ = Location parameter

Characteristic Function:
φ_NIG(u) = exp(iuμ + δ·(√(α² - β²) - √(α² - (β + iu)²)))

Moments:
Mean = μ + (δβ)/√(α² - β²)
Variance = δ·α²/(α² - β²)^{3/2}
Skewness = 3β/(α·√(δ·√(α² - β²)))

How the NIG Process Works

The NIG process combines Brownian motion with an Inverse Gaussian time change. The Inverse Gaussian subordinator creates jumps and heavy tails. The β parameter controls asymmetry: negative β creates the leverage effect. The α parameter controls tail behavior: smaller α means heavier tails. NIG often provides a better fit than Variance Gamma for equity options because of its different tail behavior.

Equity Options

Price equity options with NIG. Provides excellent fit for volatility skews and OTM option prices.

FX Derivatives

Model FX returns with NIG. Captures currency skew and fat tails in foreign exchange markets.

Parameter Estimation Guide

MLE Method

Maximum likelihood using NIG density. Requires efficient computation of Bessel functions. Use numerical optimization.

Method of Moments

Match sample mean, variance, skewness, and kurtosis. Provides good starting values for MLE.

Option Calibration

Calibrate to market option prices using FFT. Often combined with historical parameter estimates.

The NIG Discovery in Finance

Normal Inverse Gaussian distributions were introduced in physics before being applied to finance by researchers like Rydberg (1997). NIG became popular because it fit stock returns better than normal or student-t distributions.

Implementation Tips

Use scipy for Bessel functions. Implement Inverse Gaussian generator carefully. Use FFT for efficient option pricing. Validate tail behavior with QQ plots.

Frequently Asked Questions

What is the difference between NIG and Variance Gamma tails?

NIG has semi-heavy tails (exponential tails with polynomial factor). VG has heavier tails (power-law). NIG often fits financial data better for moderate tails.

Can NIG be used for multivariate option pricing?

Yes, through multivariate NIG distributions. However, multivariate extensions are complex and less commonly used in practice.

What is the Inverse Gaussian subordinator?

A continuous-time process that increases by jumps. It creates the stochastic time scale in NIG, producing the jump behavior and heavy tails.
