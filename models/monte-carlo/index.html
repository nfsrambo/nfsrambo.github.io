Quick Answer

How Monte Carlo Simulation Works

Monte Carlo simulation generates random sample paths of asset prices using Geometric Brownian Motion. For each path, compute option payoff at expiration (or intermediate times for path-dependent options). Average all payoffs and discount at risk-free rate. The Central Limit Theorem ensures that the average converges to the true value with standard error decreasing as 1/√N. Variance reduction techniques create negative correlation between paths or use known expected values as control variates to reduce estimation error.

Implementation Steps

1

Generate N random normal variables Z_i ~ N(0,1)

2

For each path, simulate price from S_0 to S_T using GBM

3

For path-dependent options, track intermediate values

4

Calculate payoff: max(S_T - K, 0) for call option

5

Sum all payoffs and divide by N for average

6

Discount average at risk-free rate: e^(-rT)

7

Calculate standard error: σ/√N

8

Report price with 95% confidence interval

Monte Carlo Applications

Asian Options

Payoff depends on average price over period. Monte Carlo naturally handles path-dependence. Need 100k+ paths for convergence.

Barrier Options

Monitor if price hits barrier during option life. Monte Carlo simulates entire path, capturing barrier events exactly.

Basket Options

Payoff depends on multiple correlated assets. Generate correlated normals using Cholesky decomposition of correlation matrix.

Risk Analysis

Simulate portfolio returns to compute Value-at-Risk (VaR) and Expected Shortfall (CVaR). Handles non-normal returns naturally.

Variance Reduction Techniques

Antithetic Variates

Use pairs (Z, -Z). Creates negative correlation, reduces variance by 2-10x. Trivial to implement, free improvement.

Control Variates

Use known option price (e.g., Black-Scholes) as control. C = C_MC + β(C_BS - C_BS_MC). Reduces variance 10-100x.

Importance Sampling

Sample more frequently from important regions (e.g., near-the-money). Adjust weights in payoff average. Critical for rare events.

Historical Development

Monte Carlo methods originated in 1940s Los Alamos for nuclear weapons research (named after Monaco casino). Phelim Boyle first applied to option pricing in 1977. The method gained prominence with computing power growth. Longstaff-Schwartz (2001) extended to American options. Today, Monte Carlo is essential for complex derivatives and risk management.

Cholesky Decomposition for Correlation

For basket options with correlation matrix Σ:

1. Compute Cholesky decomposition: Σ = LL^T
2. Generate independent normals Z_i
3. Compute correlated normals: X = LZ
4. Simulate correlated asset paths:
S_i(t+Δt) = S_i(t) * exp[(μ_i - σ_i²/2)Δt + σ_i√Δt * X_i]

Correlation captures joint price movements across assets.

Frequently Asked Questions

How do I generate correlated random numbers?

Use Cholesky decomposition of correlation matrix. Transform independent normals to correlated normals: X = LZ where Σ = LL^T. Essential for basket options and multi-asset simulation.

What random number generator should I use?

Mersenne Twister (MT19937) is standard. For high-dimensional problems, Sobol sequences (quasi-Monte Carlo) provide faster convergence. Avoid simple LCG generators.

How to price American options with Monte Carlo?

Use Least Squares Monte Carlo (Longstaff-Schwartz). At each exercise date, regress continuation value on basis functions (price, polynomial terms). Compare immediate exercise vs expected continuation.

What is the bias-variance tradeoff?

Monte Carlo has zero bias (consistent estimator) but variance decreases slowly. Variance reduction reduces standard error without introducing bias. Always prefer unbiased variance reduction.
