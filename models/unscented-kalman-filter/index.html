UKF: Unscented Kalman Filter for Finance

Master Unscented Kalman Filter for non-linear state estimation using sigma points. Learn derivative-free filtering for complex financial systems.

Quick Answer

UKF uses sigma points: 2n+1 points capture state distribution. Propagates through non-linear functions without linearization. Better than EKF for non-linear systems.

Unscented Kalman Filter Formula

Sigma Points Generation:

χ₀ = x̂
χᵢ = x̂ + (√((n+λ)P))ᵢ, i=1,...,n
χᵢ = x̂ - (√((n+λ)P))ᵢ, i=n+1,...,2n

Weights:
W₀^{(m)} = λ/(n+λ)
W₀^{(c)} = λ/(n+λ) + (1 - α² + β)
Wᵢ^{(m)} = Wᵢ^{(c)} = 1/(2(n+λ)), i=1,...,2n

Where:
λ = α²(n+κ) - n
n = State dimension
α = Sigma point spread
β = Prior distribution parameter
κ = Secondary scaling parameter

Prediction:
χ_𝒾⁻ = f(χ_𝒾)
x̂⁻ = Σ Wᵢ^{(m)}·χ_𝒾⁻
P⁻ = Σ Wᵢ^{(c)}·(χ_𝒾⁻ - x̂⁻)(χ_𝒾⁻ - x̂⁻)' + Q

How the Unscented Kalman Filter Works

The UKF uses the unscented transform, a deterministic sampling approach. It generates 2n+1 sigma points that capture the mean and covariance of the state distribution. These points are propagated through the non-linear functions, and the transformed mean and covariance are recovered. This avoids linearization and provides second-order accuracy.

Sigma Point Design Guide

Alpha Parameter

Controls sigma point spread. α ∈ [0.1, 1.0]. Smaller α means points closer to mean. Adjust for problem scale.

Beta Parameter

Incorporates prior knowledge. β=2 is optimal for Gaussian distributions. Adjust for non-Gaussian cases.

Kappa Parameter

Secondary scaling parameter. Usually 0 for state estimation. Can adjust for different problems.

The Unscented Transform Innovation

Julier and Uhlmann (1995) introduced the unscented transform and UKF. The filter improved on EKF by avoiding linearization, providing better accuracy for non-linear systems. UKF became widely used in engineering and finance.

Implementation Tips

Use square-root covariance for numerical stability. Tune α, β, κ for your problem. Validate with EKF for reference. Consider particle filter for highly non-linear cases.

Frequently Asked Questions

When to use UKF over EKF?

Use UKF for highly non-linear systems where EKF may diverge. UKF is more accurate but more computationally intensive.

How many sigma points does UKF need?

2n+1 sigma points where n is state dimension. This is deterministic and computationally efficient.

Can UKF handle non-Gaussian distributions?

UKF assumes Gaussian distributions. For non-Gaussian cases, consider particle filters or Gaussian sum filters.
