TVP-VAR: Time-Varying Parameter VAR

Master Time-Varying Parameter VAR for evolving economic relationships. Learn dynamic coefficient estimation and structural breaks in multivariate time series.

Quick Answer

TVP-VAR: Y_t = X_t·β_t + ε_t, β_t = β_{t-1} + η_t. Allows coefficients to evolve over time. Captures structural breaks and changing relationships.

TVP-VAR Formula

Measurement Equation:
Y_t = X_t·β_t + ε_t
ε_t ~ N(0, Σ_t)

State Equation:
β_t = β_{t-1} + η_t
η_t ~ N(0, Q)

Where:
Y_t = (k×1) vector of variables
X_t = (k×kp) regressors
β_t = (kp×1) time-varying coefficients
Q = State innovation covariance

Stochastic Volatility (optional):
Σ_t = A_t^{-1}·H_t·(A_t^{-1})'
h_t = h_{t-1} + ξ_t
ξ_t ~ N(0, W)

How TVP-VAR Works

TVP-VAR allows coefficients to change over time, capturing structural breaks and evolving relationships. The state-space representation uses the Kalman filter for estimation. The Bayesian approach with MCMC handles the high-dimensional parameter space.

Financial Applications

Monetary Policy Analysis

Study how monetary policy transmission changes over time. Central banks use TVP-VAR for policy evaluation.

Financial Stability Monitoring

Track changing relationships between financial variables. Identify early warning signals of instability.

Structural Break Detection

Identify periods of parameter change. Useful for understanding crises and regime shifts.

Forecasting During Crises

Improve forecasts during structural breaks. TVP-VAR adapts to changing relationships.

Prior Specification Guide

State Variance Prior

Inverse Wishart or inverse Gamma. Small variance = slow parameter change. Large variance = rapid change.

Initial State Prior

Use OLS estimate from first part of sample. Diffuse prior or informative prior based on pre-sample.

MCMC Settings

10,000-100,000 iterations. Burn-in 20-50%. Monitor convergence with Gelman-Rubin.

Primiceri's TVP-VAR Revolution

Primiceri (2005) introduced the modern TVP-VAR with stochastic volatility. The model became the standard for time-varying macroeconomics, used extensively in monetary policy analysis.

Implementation Tips

Use efficient MCMC (simulation smoother). Monitor convergence carefully. Consider stochastic volatility extension. Validate with structural break tests.

Frequently Asked Questions

What is the difference between TVP-VAR and rolling window VAR?

TVP-VAR uses a state-space framework with smooth parameter evolution. Rolling window VAR uses discrete windows with abrupt changes.

How to interpret time-varying coefficients?

They show how relationships change over time. Increases indicate stronger effects, decreases indicate weakening relationships.

What is the role of stochastic volatility in TVP-VAR?

Stochastic volatility allows error variances to change over time. Important for capturing changing uncertainty and heteroskedasticity.
