Fama-French 3-Factor Model: Asset Pricing Guide

Master Fama-French 3-factor asset pricing with market, size, and value factors. Learn factor construction, regression, and portfolio analysis.

Quick Answer

FF3: E(Rᵢ) = R_f + β_M·(R_m - R_f) + β_SMB·SMB + β_HML·HML. Adds size and value factors to CAPM. Explains 90%+ of portfolio returns.

Fama-French 3-Factor Formula

Rᵢ - R_f = α + β_M·(R_m - R_f) + β_SMB·SMB + β_HML·HML + ε

Where:
Rᵢ = Asset return
R_f = Risk-free rate
R_m = Market return
SMB = Small Minus Big (size factor)
HML = High Minus Low (value factor)
α = Jensen's alpha (excess return)
β = Factor loadings

SMB = (Small Value + Small Neutral + Small Growth)/3 - (Big Value + Big Neutral + Big Growth)/3
HML = (Small Value + Big Value)/2 - (Small Growth + Big Growth)/2

How Fama-French 3-Factor Works

The model extends CAPM by adding two factors that capture systematic risks. SMB reflects the outperformance of small-cap stocks over large-cap stocks. HML reflects the outperformance of value stocks (high book-to-market) over growth stocks (low book-to-market). These factors are constructed as long-short portfolios and represent additional sources of systematic risk.

Financial Applications

Performance Attribution

Decompose portfolio returns into factor exposure and alpha. Understand if outperformance comes from skill (α) or factor tilts.

Portfolio Construction

Target desired factor exposures. Build portfolios with specific size and value tilts to achieve return objectives.

Cost of Equity

Estimate required return for projects using factor loadings. More accurate than CAPM for companies with size/value characteristics.

Mutual Fund Analysis

Evaluate fund managers by factor-adjusted returns. Determine if alpha is due to skill or factor bets.

Factor Construction Guide

Size Breakpoints

Use NYSE median market cap to split small/big at each date. Ensures consistent breakpoints across time.

Value Breakpoints

Use bottom 30% and top 30% of book-to-market ratio for value/growth. Split at NYSE breakpoints.

Portfolio Formation

6 portfolios: 2 size × 3 value. Value-weighted returns within each. Rebalance annually in June.

The Fama-French Revolution

Fama and French (1992, 1996) revolutionized asset pricing by identifying size and value as systematic risk factors. Their work addressed anomalies that CAPM couldn't explain. The model remains the standard benchmark for evaluating portfolio performance.

Implementation Tips

Use Kenneth French's Data Library for factors. Use 36-month rolling regressions for stable beta estimates. Consider Newey-West standard errors for serial correlation. Test for factor significance and stability.

Frequently Asked Questions

What data source has Fama-French factors?

Kenneth French's Data Library provides monthly factors for US and international markets. Factors are updated regularly and freely available.

How to interpret negative factor betas?

Negative β_SMB indicates large-cap tilt. Negative β_HML indicates growth tilt. Both reduce expected returns relative to positive exposures.

Can Fama-French be used for non-US markets?

Yes. Factors can be constructed for any market. International versions show similar patterns but with different factor premiums.
